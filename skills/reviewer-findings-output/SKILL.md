## Куда писать находки

Если в промте передан `out=<путь>` — запиши полный отчёт в этот файл через Write (директорию при необходимости создай через Bash mkdir -p), а в финальном сообщении верни ровно две строки: этот путь и `находок: N`. Файл пиши всегда, даже когда находок нет (одна строка «Находок нет») — оркестратор ждёт файл, отсутствие файла трактуется как сбой.

Если `out` не передан (точечный прямой вызов) — верни находки прямо в финальном сообщении.
