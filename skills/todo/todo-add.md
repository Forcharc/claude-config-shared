# Добавить задачу в TODO список

## Аргументы
- `$ARGUMENTS` — текст задачи и опционально `@список`

## Примеры
```
/todo-add купить молоко              → inbox
/todo-add задача @работа             → work
/todo-add идея @ideas                → ideas
```

## Алгоритм

### 1. Распарси аргументы
- `@что-то` в конце → список = что-то, задача = всё до @
- Иначе → список = `inbox`

### 2. Нормализуй название списка
Преобразуй неформальное название в красивое английское:
- "работа", "рабочее" → `work`
- "личное" → `personal`
- "идеи" → `ideas`
- "покупки" → `shopping`
- и т.д.

### 3. Проверь существующие списки
Сначала запусти `python3 ~/.claude/scripts/todo_list.py --all` чтобы увидеть существующие списки.
Если есть похожий — используй его.

### 4. Выполни

```bash
python3 ~/.claude/scripts/todo_add.py LIST_NAME "TASK"
```
